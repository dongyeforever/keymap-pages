# VS Code (Windows)

> Windows-specific shortcuts for Visual Studio Code

## General

| Shortcut | Description |
|----------|-------------|
| `Ctrl+Shift+P` | Open Command Palette |
| `Ctrl+P` | Quick Open File |
| `Ctrl+K Ctrl+S` | Open Keyboard Shortcuts |
| `Alt+F4` | Close VS Code |
| `Alt+Space` | Open window menu |

## Editing

| Shortcut | Description |
|----------|-------------|
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+K Ctrl+Z` | Zen Mode (without shortcut) |
| `Ctrl+Space` | Trigger IntelliSense |
| `Ctrl+K Ctrl+I` | Show Hover |

## Navigation

| Shortcut | Description |
|----------|-------------|
| `Ctrl+Shift+\` | Jump to matching bracket |
| `Ctrl+P` then `>` | Switch to Problems panel |
| `Ctrl+P` then `@` | Go to Symbol in file |
| `Ctrl+P` then `#` | Go to Symbol in workspace |

## Window Management

| Shortcut | Description |
|----------|-------------|
| `Win+Shift+Left` | Move window to left monitor |
| `Win+Shift+Right` | Move window to right monitor |
| `Win+Up` | Maximize window |
| `Win+Down` | Minimize window |

## Integration

| Shortcut | Description |
|----------|-------------|
| `Ctrl+K Ctrl+I` | Show variable info on hover |
| `Ctrl+K Ctrl+]` | Go to next change |
| `Ctrl+K Ctrl+[` | Go to previous change |
