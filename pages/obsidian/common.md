# Obsidian

> A powerful knowledge base that works on local Markdown files

## General

| Shortcut | Description |
|----------|-------------|
| `Cmd+N` | New Note |
| `Cmd+Shift+N` | New Window |
| `Cmd+O` | Open Folder |
| `Cmd+S` | Save |
| `Cmd+Shift+S` | Save and Sync |
| `Cmd+W` | Close Tab |
| `Cmd+P` | Quick Switcher |
| `Cmd+Shift+P` | Command Palette |
| `Cmd+,` | Settings |
