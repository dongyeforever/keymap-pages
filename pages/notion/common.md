# Notion

> All-in-one workspace for notes, docs, and project management

## General

| Shortcut | Description |
|----------|-------------|
| `Cmd+N` | New Page |
| `Cmd+Shift+N` | New Page in Sidebar |
| `Cmd+P` | Quick Find |
| `Cmd+Shift+P` | Go to Page |
| `Cmd+\` | Toggle Sidebar |
| `Cmd+Shift+L` | Toggle List View |
| `Cmd+Shift+>` | Toggle Board View |
| `Cmd+Shift+1` | Add Icon |
| `Cmd+Shift+2` | Add Cover |
| `Cmd+Shift+H` | Turn Into |
| `Cmd+Shift+D` | Duplicate |
| `Cmd+Shift+Del` | Delete |
