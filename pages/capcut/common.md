# CapCut

> Free video editing app by ByteDance

## General

| Shortcut | Description |
|----------|-------------|
| `Cmd+N` | New Project |
| `Cmd+O` | Open Project |
| `Cmd+S` | Save |
| `Cmd+Shift+S` | Save As |
| `Cmd+Z` | Undo |
| `Cmd+Shift+Z` | Redo |
| `Cmd+C` | Copy |
| `Cmd+V` | Paste |
| `Cmd+X` | Cut |
| `Cmd+D` | Duplicate |
| `Cmd+A` | Select All |
| `Delete` | Delete |
| `Backspace` | Delete |
