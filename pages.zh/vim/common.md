# Vim

> 终端文本编辑器

## 基本模式

| 命令 | 说明 |
|------|------|
| `i` | 进入插入模式 |
| `Esc` | 退出插入模式，回到普通模式 |
| `v` | 进入可视模式 |
| `V` | 进入可视行模式 |
| `Ctrl+v` | 进入可视块模式 |
| `:` | 进入命令行模式 |
| `R` | 进入替换模式 |

## 基本移动

| 命令 | 说明 |
|------|------|
| `h` | 左移 |
| `j` | 下移 |
| `k` | 上移 |
| `l` | 右移 |
| `w` | 移动到下一个单词开头 |
| `b` | 移动到上一个单词开头 |
| `e` | 移动到单词结尾 |
| `0` | 移动到行首 |
| `$` | 移动到行尾 |
| `^` | 移动到行首（非空白） |
| `gg` | 移动到文件开头 |
| `G` | 移动到文件结尾 |
| `:n` | 移动到第 n 行 |
| `nG` | 移动到第 n 行 |

## 搜索

| 命令 | 说明 |
|------|------|
| `/pattern` | 向下搜索 |
| `?pattern` | 向上搜索 |
| `n` | 下一个匹配 |
| `N` | 上一个匹配 |
| `*` | 搜索光标下的词 |
| `#` | 反向搜索光标下的词 |
| `:noh` | 取消高亮 |

## 编辑

| 命令 | 说明 |
|------|------|
| `x` | 删除光标处字符 |
| `dd` | 删除整行 |
| `dw` | 删除单词 |
| `d$` | 删除到行尾 |
| `d0` | 删除到行首 |
| `yy` | 复制整行 |
| `yw` | 复制单词 |
| `y$` | 复制到行尾 |
| `p` | 粘贴到光标后 |
| `P` | 粘贴到光标前 |
| `u` | 撤销 |
| `Ctrl+r` | 重做 |
| `.` | 重复上次操作 |
| `cc` | 修改整行 |
| `cw` | 修改单词 |
| `c$` | 修改到行尾 |

## 文本对象

| 命令 | 说明 |
|------|------|
| `iw` | 内部单词 |
| `aw` | 一个单词（含空格） |
| `i"` | 引号内 |
| `a"` | 含引号 |
| `i(` `i)` | 括号内 |
| `a(` `a)` | 含括号 |
| `i[` `i]` | 方括号内 |
| `i{` `i}` | 大括号内 |

## 窗口操作

| 命令 | 说明 |
|------|------|
| `:sp` | 水平分屏 |
| `:vsp` | 垂直分屏 |
| `Ctrl+w h` | 切换到左窗口 |
| `Ctrl+w j` | 切换到下窗口 |
| `Ctrl+w k` | 切换到上窗口 |
| `Ctrl+w l` | 切换到右窗口 |
| `Ctrl+w w` | 循环切换窗口 |
| `Ctrl+w =` | 所有窗口等高 |
| `Ctrl+w _` | 最大化高度 |
| `Ctrl+w |` | 最大化宽度 |
| `:q` | 关闭窗口 |
| `:qa` | 关闭所有窗口 |

## 缓冲区

| 命令 | 说明 |
|------|------|
| `:e filename` | 打开文件 |
| `:bnext` | 下一个缓冲区 |
| `:bprevious` | 上一个缓冲区 |
| `:bdelete` | 关闭缓冲区 |
| `:buffers` | 列出缓冲区 |

## 书签

| 命令 | 说明 |
|------|------|
| `m[a-z]` | 设置书签 a-z |
| `` `[a-z]`` | 跳转到书签 |
| `:marks` | 列出书签 |

## 宏

| 命令 | 说明 |
|------|------|
| `q[a-z]` | 开始录制宏 |
| `q` | 停止录制 |
| `@[a-z]` | 执行宏 |
| `@@` | 重复上次宏 |

## 命令行

| 命令 | 说明 |
|------|------|
| `:w` | 保存 |
| `:q` | 退出 |
| `:wq` / `:x` | 保存并退出 |
| `:q!` | 不保存强制退出 |
| `:s/old/new` | 替换 |
| `:s/old/new/g` | 全部替换 |
| `:%s/old/new/g` | 全局替换 |
| `:r filename` | 插入文件内容 |
| `:!command` | 执行外部命令 |

## 快捷键组合

| 命令 | 说明 |
|------|------|
| `>>` | 右缩进 |
| `<<` | 左缩进 |
| `==` | 自动缩进 |
| `gUU` | 全行大写 |
| `guu` | 全行小写 |
| `g~w` | 大小写切换 |
| `gUiw` | 单词大写 |
| `guiw` | 单词小写 |
