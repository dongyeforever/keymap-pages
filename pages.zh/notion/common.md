# Notion

> 集成笔记、文档和项目管理的全能工作空间

## 通用

| 快捷键 | 说明 |
|--------|------|
| `Cmd+N` | 新建页面 |
| `Cmd+Shift+N` | 在侧边栏新建页面 |
| `Cmd+P` | 快速查找 |
| `Cmd+Shift+P` | 跳转到页面 |
| `Cmd+\` | 切换侧边栏 |
| `Cmd+Shift+L` | 切换列表视图 |
| `Cmd+Shift+>` | 切换看板视图 |
| `Cmd+Shift+1` | 添加图标 |
| `Cmd+Shift+2` | 添加封面 |
| `Cmd+Shift+H` | 转换为 |
| `Cmd+Shift+D` | 复制 |
| `Cmd+Shift+Del` | 删除 |
