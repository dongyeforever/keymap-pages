# Android Studio

> Android 应用开发官方 IDE

## 通用

| 快捷键 | 说明 |
|--------|------|
| `Cmd+N` | 新建 |
| `Cmd+O` | 打开 |
| `Cmd+S` | 保存 |
| `Cmd+Shift+S` | 全部保存 |
| `Cmd+Alt+S` | 项目结构 |
| `Cmd+,` | 设置 |
| `Cmd+;` | 项目设置 |
| `Cmd+Q` | 退出 |
