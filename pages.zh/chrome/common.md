# Chrome

> Google 浏览器

## 标签页和窗口

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+T` | 打开新标签页 |
| `Ctrl+Shift+T` | 重新打开关闭的标签页 |
| `Ctrl+W` / `Ctrl+F4` | 关闭当前标签页 |
| `Ctrl+Shift+W` | 关闭当前窗口 |
| `Ctrl+Tab` / `Ctrl+Page Down` | 切换到下一个标签页 |
| `Ctrl+Page Up` | 切换到上一个标签页 |
| `Ctrl+1` 到 `Ctrl+8` | 切换到第 1-8 个标签页 |
| `Ctrl+9` | 切换到最后一个标签页 |
| `Ctrl+Shift+Tab` / `Ctrl+Page Up` | 切换到上一个标签页 |
| `Ctrl+N` | 打开新窗口 |
| `Ctrl+Shift+N` | 打开新的隐身窗口 |

## 页面导航

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+L` / `Alt+D` | 聚焦到地址栏 |
| `Enter` | 转到地址 |
| `Alt+←` | 后退 |
| `Alt+→` | 前进 |
| `F5` / `Ctrl+R` | 重新加载 |
| `Ctrl+Shift+R` | 强制重新加载 |
| `Escape` | 停止加载 |
| `Home` | 转到页面顶部 |
| `End` | 转到页面底部 |
| `Space` | 向下滚动一页 |
| `Shift+Space` | 向上滚动一页 |
| `Page Down` | 向下滚动一页 |
| `Page Up` | 向上滚动一页 |
| `↑` / `↓` | 向上/向下滚动 |

## 书签

## 页面功能

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+P` | 打印页面 |
| `Ctrl+S` | 保存页面 |
| `Ctrl+U` | 查看页面源代码 |

## 搜索

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+K` / `Ctrl+E` | 聚焦到搜索框 |
| `Ctrl+F` / `F3` | 页面内查找 |
| `Ctrl+G` | 查找下一个 |
| `Ctrl+Shift+G` | 查找上一个 |

## 标签页操作

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+Shift+T` | 恢复关闭的标签页 |
| `Alt+Enter` | 在新标签页中打开搜索结果 |
| `Ctrl+Click` | 在后台打开链接 |
| `Ctrl+Shift+Click` | 在新前台打开链接 |
| `Middle Click` | 在新标签页打开链接 |
