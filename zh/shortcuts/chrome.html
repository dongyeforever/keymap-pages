Chrome ​

Google 浏览器

标签页和窗口 ​

快捷键说明
Ctrl+T打开新标签页
Ctrl+Shift+T重新打开关闭的标签页
Ctrl+W / Ctrl+F4关闭当前标签页
Ctrl+Shift+W关闭当前窗口
Ctrl+Tab / Ctrl+Page Down切换到下一个标签页
Ctrl+Page Up切换到上一个标签页
Ctrl+1 到 Ctrl+8切换到第 1-8 个标签页
Ctrl+9切换到最后一个标签页
Ctrl+Shift+Tab / Ctrl+Page Up切换到上一个标签页
Ctrl+N打开新窗口
Ctrl+Shift+N打开新的隐身窗口

页面导航 ​

快捷键说明
Ctrl+L / Alt+D聚焦到地址栏
Enter转到地址
Alt+←后退
Alt+→前进
F5 / Ctrl+R重新加载
Ctrl+Shift+R强制重新加载
Escape停止加载
Home转到页面顶部
End转到页面底部
Space向下滚动一页
Shift+Space向上滚动一页
Page Down向下滚动一页
Page Up向上滚动一页
↑ / ↓向上/向下滚动

书签 ​

快捷键说明
Ctrl+D添加当前页面到书签
Ctrl+Shift+B显示/隐藏书签栏
Ctrl+Shift+O打开书签管理器

页面功能 ​

快捷键说明
Ctrl+P打印页面
Ctrl+S保存页面
Ctrl+U查看页面源代码
F12 / Ctrl+Shift+I打开开发者工具
Ctrl+Shift+J打开控制台
Ctrl+Shift+M打开设备模式
Ctrl++放大
Ctrl+-缩小
Ctrl+0重置缩放

搜索 ​

快捷键说明
Ctrl+K / Ctrl+E聚焦到搜索框
Ctrl+F / F3页面内查找
Ctrl+G查找下一个
Ctrl+Shift+G查找上一个
Escape关闭查找栏

标签页操作 ​

快捷键说明
Ctrl+Shift+T恢复关闭的标签页
Alt+Enter在新标签页中打开搜索结果
Ctrl+Click在后台打开链接
Ctrl+Shift+Click在新前台打开链接
Middle Click在新标签页打开链接

下载和历史 ​

快捷键说明
Ctrl+J打开下载内容
Ctrl+H打开历史记录
Ctrl+Shift+Delete打开清除浏览数据

开发者工具 ​

快捷键说明
F12打开 DevTools
Ctrl+Shift+I打开 DevTools
Ctrl+Shift+J打开控制台面板
Ctrl+[切换到上一面板
Ctrl+]切换到下一面板
Ctrl+Shift+P快速切换主题
Ctrl+Shift+C检查元素模式
