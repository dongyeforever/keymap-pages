# IntelliJ IDEA

> JetBrains 出品的 Java 和 Kotlin 集成开发环境

## 通用

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+Shift+P` | 打开命令查找器 |
| `Ctrl+Shift+A` | 动作搜索 |
| `Shift+Shift` | 快速搜索 everywhere |
| `Ctrl+Alt+S` | 打开设置对话框 |
| `Ctrl+Alt+Shift+S` | 打开项目结构 |
| `Ctrl+,` | 打开快速设置 |
| `F1` | 查看文档 |

## 编辑

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+C` | 复制（空选择时复制整行） |
| `Ctrl+X` | 剪切（空选择时剪切整行） |
| `Ctrl+V` | 粘贴 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
| `Ctrl+D` | 复制当前行或选择 |
| `Ctrl+Y` | 删除当前行 |
| `Alt+Shift+Up` | 向上移动行 |
| `Alt+Shift+Down` | 向下移动行 |
| `Ctrl+/` | 切换行注释 |
| `Ctrl+Shift+/` | 切换块注释 |
| `Ctrl+Alt+L` | 格式化代码 |
| `Ctrl+Alt+I` | 自动缩进 |
| `Ctrl+Shift+Up` | 向上移动语句 |
| `Ctrl+Shift+Down` | 向下移动语句 |
