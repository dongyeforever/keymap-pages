# iTerm2

> Terminal emulator for macOS

## General

| Shortcut | Description |
|----------|-------------|
| `Cmd+N` | New Tab |
| `Cmd+Shift+N` | New Window |
| `Cmd+T` | New Tab with Current Directory |
| `Cmd+W` | Close Tab |
| `Cmd+Shift+W` | Close Window |
| `Cmd+Q` | Quit iTerm2 |
| `Cmd+,` | Preferences |
| `Cmd+Opt+1` | Switch Tab 1 |
| `Cmd+Opt+2` | Switch Tab 2 |
| `Cmd+Opt+3` | Switch Tab 3 |
| `Cmd+Opt+4` | Switch Tab 4 |
| `Cmd+Opt+5` | Switch Tab 5 |
| `Cmd+Opt+6` | Switch Tab 6 |
| `Cmd+Opt+7` | Switch Tab 7 |
| `Cmd+Opt+8` | Switch Tab 8 |
| `Cmd+Opt+9` | Switch Tab 9 |
| `Cmd+Opt+Left` | Previous Tab |
| `Cmd+Opt+Right` | Next Tab |
| `Cmd+Shift+[` | Previous Tab |
| `Cmd+Shift+]` | Next Tab |

## Navigation

| Shortcut | Description |
|----------|-------------|
| `Cmd+Arrow` | Move by word (with option) |
| `Cmd+Click` | Open URL/File |
| `Cmd+Scroll` | Search Scroll |
| `Ctrl+A` | Move to beginning of line |
| `Ctrl+E` | Move to end of line |
| `Ctrl+U` | Clear line |
| `Ctrl+K` | Clear to end of line |
| `Ctrl+L` | Clear screen |
| `Ctrl+W` | Delete word |
| `Ctrl+Y` | Paste (last yank) |

|----------|-------------|

| `Cmd+V` | Paste |

| `Cmd+Shift+V` | Selective Paste |
| `Cmd+K` | Clear screen |
| `Cmd+Shift+K` | Clear scrollback |
| `Cmd+T` | Open URL/File |
| `Cmd+Enter` | Mark Prompt |

| `Cmd+Opt+Up` | Go to Upper Split |
| `Cmd+Opt+Down` | Go to Lower Split |
| `Cmd+Shift+[` | Previous Split |
| `Cmd+Shift+]` | Next Split |
