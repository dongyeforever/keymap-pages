# iTerm2

> macOS 终端模拟器

## 通用

| 快捷键 | 说明 |
|--------|------|
| `Cmd+N` | 新建标签页 |
| `Cmd+Shift+N` | 新建窗口 |
| `Cmd+T` | 在当前目录新建标签页 |
| `Cmd+W` | 关闭标签页 |
| `Cmd+Shift+W` | 关闭窗口 |
| `Cmd+Q` | 退出 iTerm2 |
| `Cmd+,` | 偏好设置 |
| `Cmd+Opt+1` | 切换到标签 1 |
| `Cmd+Opt+2` | 切换到标签 2 |
| `Cmd+Opt+3` | 切换到标签 3 |
| `Cmd+Opt+4` | 切换到标签 4 |
| `Cmd+Opt+5` | 切换到标签 5 |
| `Cmd+Opt+6` | 切换到标签 6 |
| `Cmd+Opt+7` | 切换到标签 7 |
| `Cmd+Opt+8` | 切换到标签 8 |
| `Cmd+Opt+9` | 切换到标签 9 |
| `Cmd+Opt+Left` | 上一个标签页 |
| `Cmd+Opt+Right` | 下一个标签页 |
| `Cmd+Shift+[` | 上一个标签页 |
| `Cmd+Shift+]` | 下一个标签页 |

## 导航

| 快捷键 | 说明 |
|--------|------|
| `Cmd+Arrow` | 按词移动（配合 option） |
| `Cmd+Click` | 打开 URL/文件 |
| `Cmd+Scroll` | 搜索滚动 |
| `Ctrl+A` | 移动到行首 |
| `Ctrl+E` | 移动到行尾 |
| `Ctrl+U` | 清除当前行 |
| `Ctrl+K` | 清除到行尾 |
| `Ctrl+L` | 清屏 |
| `Ctrl+W` | 删除单词 |
| `Ctrl+Y` | 粘贴（上次复制） |

## 文本操作

| 快捷键 | 说明 |
|--------|------|
| `Cmd+C` | 复制 |
| `Cmd+V` | 粘贴 |
| `Cmd+Shift+C` | 带提示符复制 |
| `Cmd+Shift+V` | 选择性粘贴 |
| `Cmd+K` | 清屏 |
| `Cmd+Shift+K` | 清除滚动历史 |
| `Cmd+T` | 打开 URL/文件 |
| `Cmd+Enter` | 标记当前位置 |
| `Cmd+Shift+Enter` | 跳转到标记 |

## 分屏

| 快捷键 | 说明 |
|--------|------|
| `Cmd+D` | 垂直分屏 |
| `Cmd+Shift+D` | 水平分屏 |
| `Cmd+Opt+Arrow` | 切换分屏 |
| `Cmd+Opt+Left` | 切换到左侧分屏 |
| `Cmd+Opt+Right` | 切换到右侧分屏 |
| `Cmd+Opt+Up` | 切换到上方分屏 |
| `Cmd+Opt+Down` | 切换到下方分屏 |
| `Cmd+Shift+[` | 上一个分屏 |
| `Cmd+Shift+]` | 下一个分屏 |

## 搜索

| 快捷键 | 说明 |
|--------|------|
| `Cmd+F` | 搜索 |
| `Cmd+G` | 查找下一个 |
| `Cmd+Shift+G` | 查找上一个 |
| `Cmd+Opt+F` | 在历史中搜索 |
| `Cmd+Shift+H` | 显示历史 |

## 配置

| 快捷键 | 说明 |
|--------|------|
| `Cmd+Opt+S` | 切换智能选择 |
| `Cmd+Shift+S` | SSH/telnet |
| `Cmd+Shift+O` | 编辑当前会话 |
| `Cmd+Opt+E` | 编辑配置 |
| `Cmd+Shift+A` | 新建配置 |

## 窗口

| 快捷键 | 说明 |
|--------|------|
| `Cmd+Enter` | 全屏 |
| `Cmd+Opt+F` | 切换全屏 |
| `Cmd+Opt+Z` | 放大 |
| `Cmd+Shift+Z` | 缩小 |
| `Cmd+M` | 最小化 |
| `Cmd+H` | 隐藏 |
| `Cmd+Option+H` | 隐藏其他 |

## 快速查看

| 快捷键 | 说明 |
|--------|------|
| `Cmd+Shift+Space` | 快速查看 |
