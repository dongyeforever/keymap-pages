# Figma

> 协作式界面设计工具

## 工具

| 快捷键 | 说明 |
|--------|------|
| `V` | 移动工具 |
| `F` | 帧工具 |
| `R` | 矩形工具 |
| `O` | 椭圆工具 |
| `L` | 线条工具 |
| `T` | 文本工具 |
| `P` | 钢笔工具 |
| `H` | 手型工具（临时） |
| `I` | 取色器 |
| `Ctrl+G` | 编组 |
| `Ctrl+Shift+G` | 取消编组 |
| `Ctrl+Alt+K` | 创建组件 |

## 编辑

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+C` | 复制 |
| `Ctrl+V` | 粘贴 |
| `Ctrl+D` | 原地复制 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
| `Delete` | 删除 |
| `Ctrl+Shift+D` | 再次分布 |
| `Ctrl+Alt+I` | 水平镜像复制 |
| `Ctrl+Shift+I` | 垂直镜像复制 |

## 选择

| 快捷键 | 说明 |
|--------|------|
| `Esc` | 取消选择 |
| `Enter` | 进入编组/帧 |
| `Tab` | 进入下一个编组 |
| `Shift+Tab` | 进入上一个编组 |
| `Ctrl+A` | 全选 |
| `Ctrl+Shift+A` | 反向选择 |
| `Ctrl+Click` | 添加到选择 |
| `Ctrl+Drag` | 多重选择 |
| `Alt+Hover` | 查看层级 |
| `Alt+Click` | 选中父级 |

## 画布操作

| 快捷键 | 说明 |
|--------|------|
| `Space+Drag` | 平移画布 |
| `Ctrl++` | 放大 |
| `Ctrl+-` | 缩小 |
| `Ctrl+0` | 适应屏幕 |
| `Ctrl+1` | 缩放到 100% |
| `Ctrl+2` | 缩放到选区 |
| `Shift+1` | 视图：画板 |
| `Shift+2` | 视图：画板内容 |
