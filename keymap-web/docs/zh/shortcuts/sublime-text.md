# Sublime Text

> 专业的代码、标记和散文编辑器

## 通用

| 快捷键 | 说明 |
|--------|------|
| `Cmd+N` | 新建文件 |
| `Cmd+O` | 打开文件 |
| `Cmd+Shift+N` | 新建窗口 |
| `Cmd+S` | 保存 |
| `Cmd+Shift+S` | 另存为 |
| `Cmd+W` | 关闭文件 |
| `Cmd+Shift+W` | 关闭窗口 |
| `Cmd+Q` | 退出 |
| `Cmd+,` | 设置 |
| `Cmd+Opt+S` | 全部保存 |

## 编辑

| 快捷键 | 说明 |
|--------|------|
| `Cmd+Z` | 撤销 |
| `Cmd+Shift+Z` | 重做 |
| `Cmd+C` | 复制 |
| `Cmd+V` | 粘贴 |
| `Cmd+X` | 剪切 |
| `Cmd+Shift+V` | 从历史粘贴 |
| `Cmd+Alt+Shift+Up` | 向上添加列选择 |
| `Cmd+Alt+Shift+Down` | 向下添加列选择 |
| `Cmd+D` | 添加下一个选择 |
| `Cmd+Shift+L` | 将选区分成行 |
| `Cmd+Shift+D` | 复制选区 |
| `Cmd+J` | 合并行 |
| `Cmd+Enter` | 在下方插入行 |
| `Cmd+Shift+Enter` | 在上方插入行 |
