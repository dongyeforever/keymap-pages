# Typora

> 极简 Markdown 编辑器

## 通用

| 快捷键 | 说明 |
|--------|------|
| `Cmd+N` | 新建文件 |
| `Cmd+O` | 打开文件 |
| `Cmd+S` | 保存 |
| `Cmd+Shift+S` | 另存为 |
| `Cmd+W` | 关闭 |
| `Cmd+Q` | 退出 |
| `Cmd+,` | 偏好设置 |
| `Cmd+Opt+S` | 全部保存 |
