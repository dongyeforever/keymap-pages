# Obsidian

> 基于本地 Markdown 文件的强大知识库

## 通用

| 快捷键 | 说明 |
|--------|------|
| `Cmd+N` | 新建笔记 |
| `Cmd+Shift+N` | 新建窗口 |
| `Cmd+O` | 打开文件夹 |
| `Cmd+S` | 保存 |
| `Cmd+Shift+S` | 保存并同步 |
| `Cmd+W` | 关闭标签页 |
| `Cmd+P` | 快速切换 |
| `Cmd+Shift+P` | 命令面板 |
| `Cmd+,` | 设置 |

## 编辑

| 快捷键 | 说明 |
|--------|------|
| `Cmd+Z` | 撤销 |
| `Cmd+Shift+Z` | 重做 |
| `Cmd+B` | 粗体 |
| `Cmd+I` | 斜体 |
| `Cmd+U` | 下划线 |
| `Cmd+Shift+K` | 取消链接 |
| `Cmd+Shift+H` | 高亮 |
| `Cmd+E` | 切换代码块 |
| `Cmd+~` | 切换引用块 |
| `Cmd+D` | 复制 |

## 格式化

| 快捷键 | 说明 |
|--------|------|
| `Cmd+1` | 一级标题 |
| `Cmd+2` | 二级标题 |
| `Cmd+3` | 三级标题 |
| `Cmd+4` | 四级标题 |
| `Cmd+5` | 五级标题 |
| `Cmd+6` | 六级标题 |
| `Cmd+0` | 正文 |
| `Cmd+Shift+1` | 无序列表 |
| `Cmd+Shift+2` | 有序列表 |
| `Cmd+Shift+3` | 任务列表 |
| `Cmd+Shift+4` | 复选框 |
| `Cmd+Shift+5` | 引用 |
| `Cmd+Shift+7` | 代码 |
| `Cmd+Shift+8` | 提示块 |
| `/` | 插入块 |

## 链接

| 快捷键 | 说明 |
|--------|------|
| `Cmd+K` | 插入链接 |
| `Cmd+Shift+U` | 插入 URL |
| `Cmd+Option+E` | 嵌入文件 |
| `Cmd+Shift+I` | 从路径插入图片 |

## 导航

| 快捷键 | 说明 |
|--------|------|
| `Cmd+Option+Left` | 后退 |
| `Cmd+Option+Right` | 前进 |
| `Cmd+Click` | 在新窗口打开 |
| `Cmd+Shift+Click` | 在新窗口打开 |
| `Cmd+Alt+Click` | 在右侧窗口打开 |

## 文件树

| 快捷键 | 说明 |
|--------|------|
| `Cmd+Alt+F` | 文件搜索 |
| `Cmd+F` | 在当前文件中查找 |
| `Cmd+Shift+F` | 在所有文件中查找 |
| `Cmd+G` | 跳转到标题 |
| `Cmd+Shift+G` | 跳转到日期 |

## 分屏

| 快捷键 | 说明 |
