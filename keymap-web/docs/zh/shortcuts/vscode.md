# VS Code

> Visual Studio Code - 跨平台代码编辑器

## 常用

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+Shift+P` | 打开命令面板 |
| `Ctrl+P` | 快速打开文件 |
| `Ctrl+Shift+N` | 新建窗口 |
| `Ctrl+W` | 关闭窗口 |
| `Ctrl+K Ctrl+S` | 打开快捷键设置 |
| `Ctrl+,` | 打开设置 |
| `F1` | 显示所有命令 |

## 编辑

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+C` | 复制行（空选区时） |
| `Ctrl+X` | 剪切行（空选区时） |
| `Ctrl+V` | 粘贴 |
| `Ctrl+Shift+K` | 删除行 |
| `Alt+Up` | 上移行 |
| `Alt+Down` | 下移行 |
| `Ctrl+Shift+Alt+Up` | 向上复制行 |
| `Ctrl+Shift+Alt+Down` | 向下复制行 |
| `Ctrl+/` | 切换行注释 |
| `Ctrl+Shift+A` | 切换块注释 |
| `Ctrl+Shift+[` | 折叠代码块 |
| `Ctrl+Shift+]` | 展开代码块 |
| `Ctrl+K Ctrl+L` | 折叠所有区域 |
| `Ctrl+K Ctrl+J` | 展开所有区域 |

## 导航

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+G` | 跳转到行 |
| `Ctrl+Shift+O` | 跳转到符号 |
| `Ctrl+Tab` | 切换打开的文件 |
| `Ctrl+Shift+Tab` | 上一个打开的文件 |
| `Ctrl+Shift+\` | 跳转到匹配的括号 |
| `F12` | 跳转到定义 |
| `Alt+F12` | .peek 定义 |
| `Ctrl+Click` | 打开定义 |
| `Ctrl+Shift+F12` | .peek 引用 |

## 搜索和替换

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+F` | 查找 |
| `Ctrl+H` | 替换 |
| `Ctrl+Shift+F` | 在文件中查找 |
| `Ctrl+Shift+H` | 在文件中替换 |
| `F3` | 查找下一个 |
| `Shift+F3` | 查找上一个 |
| `Alt+Enter` | 选中所有匹配项 |

## 多光标和选区

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+Alt+Down` | 在下方添加光标 |
| `Ctrl+Alt+Up` | 在上方添加光标 |
| `Ctrl+Shift+L` | 选中所有匹配项 |
| `Ctrl+D` | 将选区添加到下一个匹配 |
| `Ctrl+Shift+Space` | 触发 IntelliSense 建议 |
| `Shift+Arrow` | 选择文本 |
| `Ctrl+Shift+Arrow` | 词选择 |
