# CapCut

> 字节跳动推出的免费视频编辑应用

## 通用

| 快捷键 | 说明 |
|--------|------|
| `Cmd+N` | 新建项目 |
| `Cmd+O` | 打开项目 |
| `Cmd+S` | 保存 |
| `Cmd+Shift+S` | 另存为 |
| `Cmd+Z` | 撤销 |
| `Cmd+Shift+Z` | 重做 |
| `Cmd+C` | 复制 |
| `Cmd+V` | 粘贴 |
| `Cmd+X` | 剪切 |
| `Cmd+D` | 复制 |
| `Cmd+A` | 全选 |
| `Delete` | 删除 |
| `Backspace` | 删除 |

## 播放

| 快捷键 | 说明 |
|--------|------|
| `Space` | 播放/暂停 |
| `J` | 倒放 |
| `L` | 加速播放 |
| `K` | 停止 |
| `←` | 上一帧 |
| `→` | 下一帧 |
| `Shift+←` | 后退 |
| `Shift+→` | 前进 |
| `I` | 设置入点 |
| `O` | 设置出点 |

| `Cmd+K` | 剪切-clip |
| `[` | 移动到开头 |
| `]` | 移动到结尾 |

| `Down` | 右移 |
